# Changelog

Все заметные изменения в проекте iikoServiceHelper будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- **AppConstants**: Введен класс констант для устранения магических чисел и строк.
- **DebounceDispatcher**: Добавлена утилита для оптимизации частых операций (автосохранение).
- **SaveNotesAsync/SaveSettingsAsync**: Асинхронные методы сохранения для предотвращения блокировки UI.
- **.editorconfig**: Добавлен файл конфигурации стиля кода.
- **XML Documentation**: Добавлена документация для ключевых классов и методов.
- **Workflows**: Добавлены инструкции по улучшению проекта в `.agent/workflows`.

### Changed

- **CrmAutoLoginService**:
  - Порт 9222 заменен на константу.
  - Добавлен `ConfigureAwait(false)` для всех асинхронных вызовов.
- **MainWindow**:
  - Автосохранение заметок теперь использует Debounce (1 сек) и асинхронную запись.
  - Сохранение настроек при потере фокуса теперь асинхронно.
  - Имена файлов и настройки переведены на константы.
- **Refactoring**: Удалены хардкодные значения порта 9222, таймаутов и путей к файлам.

### Fixed

- Исправлено потенциальное зависание UI при сохранении больших заметок.
- Улучшена читаемость кода за счет использования констант.
